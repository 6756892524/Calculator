Simple Calculator Web App :

A lightweight and responsive web-based calculator developed using HTML, CSS, and JavaScript to perform basic arithmetic operations with real-time user input handling.

--Features
1.Addition, Subtraction, Multiplication, Division

2.Clean and responsive UI

3.Instant result calculation

4.Error handling for invalid inputs

--Tech Stack

1.HTML5 – Structure

2.CSS3 – Styling & Layout

3.JavaScript (ES6) – Logic & Interactivity

--Usage

1.Clone or download the repository

2.Open index.html in any web browser

3.Project Structure
/calculator
  ├─ index.html
  ├─ style.css
  └─ script.js

--Key Learning Outcomes

1.DOM Manipulation

2.Event Handling

3.Responsive Design Principles

4.JavaScript Logic Building
